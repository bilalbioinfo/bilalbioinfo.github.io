Welcome, and thanks for stopping by. This is the first post on my new site.

I plan to use this space to write, from time to time, about the things I work on day to day: **bioinformatics for ancient and degraded DNA**, reproducible pipelines (mostly [Nextflow](https://www.nextflow.io/)), read-mapping and variant-calling strategies for damaged sequences, target-capture design, and the occasional note on tools and workflows I find useful.

Posts will be a mix of short technical notes, tutorials, and reflections on doing computational research. To write a new post, add a Markdown file to the `_posts/` folder named `YYYY-MM-DD-title.md` — this file is a working example you can copy.

More soon.
